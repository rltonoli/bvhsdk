# Usage

## Installation

git clone the bvhsdk repository from https://github.com/rltonoli/bvhsdk or install it using pip

```python
pip install bvhsdk
```

## Open BVH file

```python
import bvhsdk

anim = bvhsdk.ReadFile('pathtobvh.bvh')
```
